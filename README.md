Use boto3 to manage EC2 instances.

## Install

```sh
pipenv install
```

```sh
aws configure --profile shotty
```


## Run

```sh
pipenv run python shotty/shotty.py --help

```

## Distributions

```sh
pip3 install https://jex-python-dist.s3.amazonaws.com/snapshotalyzer_30000-1.0-py3-none-any.whl
```
